# E-Commerce Sales Analytics Dashboard

## Project Overview
This project analyzes e-commerce sales data to understand revenue trends, customer behavior, and product performance.  
The project uses Excel for data cleaning, SQL for analysis, and Power BI for visualization.

## Tools Used
- Excel (Data Cleaning)
- SQL (Data Analysis)
- Power BI (Dashboard Visualization)

## Key Insights
- Total Revenue: 8.28M
- Total Orders: 22K
- Total Customers: 4,372
- Average Order Value: 373
- United Kingdom is the top revenue-generating country
- Strong revenue growth in the last quarter
- 3,059 repeat customers indicate good customer retention

## Business Recommendations
- Improve customer retention through loyalty programs
- Focus marketing on high-performing countries
- Promote sales during low-revenue months
- Keep top-selling products well stocked
